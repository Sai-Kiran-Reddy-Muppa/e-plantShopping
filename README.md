# e-plantShopping

e-plantShopping is a React-based plant shopping application that allows users to browse plants by category, add items to a shopping cart, and manage quantities using Redux.

## Features
- Landing page with background image
- Product listing grouped by categories
- Add to cart functionality using Redux
- Cart page with quantity updates and total calculation
- React Router navigation

## Technologies Used
- React
- Redux Toolkit
- React Router
- CSS
